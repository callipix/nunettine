프로젝트 소개

프로젝트 이름의 의미
'누네띠네'는 생활서비스를 필요로 하는 이용자들 어디에서나 쉽게 눈에 띌수 있었으면 하는 바람으로 이름지었습니다.

프로젝트 간단 요약
누네띠네는 생활서비스 전문가 매칭/제공 시스템으로써 일상 생활속에서 필요한 생활서비스에 대한 전문가(이하 프로)들을
검색하고 해당 서비스에 대한 견적과 서비스를 제공받는 웹서비스입니다. 기존에 존재하는 숨고 라는 서비스 매칭 플랫폼을 벤치마킹 하여
우리만의 스타일로 재해석하였고 누구나 쉽게 접근할 수 있도록 간편한 접근성과 편리성, 이로 인해 프로는 고부가가치 창출
회원은 합리적인 가격으로 서비스를 이용하여 경제성을 추구하고자 하였으며, 회원/프로들 간의 모임을 통한 커뮤니티 활동도 가능합니다.

프로젝트 기능 소개
프로찾기 : 프로를 검색
프로 서비스문의/요청 : 해당 프로들에 대한 서비스문의/요청
원데이클래스 : 프로들이 본인의 해당 영역에서 일일 수업으로 가르치는 수업
후기작성 : 프로와 서비스 이용 회원이 제공한/받은 요청/서비스 완료 여부에 따른 후기작성
자유게시판 : 일반적인 형태의 게시판
오늘모임 : 회원/프로간의 커뮤니티적 요소를 위한 자유로운 단발성 모임을 생성
오늘모임 - 채팅 : 오늘모임에서의 기능으로써 모임을 생성하면 기본적으로 존재하는 채팅방, 자유로운 주제로 실시간 채팅가능
프로이야기 : 프로가 본인만의 에피소드 혹은 자유주제를 가지고 이야기 형식으로 풀어내어 자신을 소개/홍보하는 게시판
관리자페이지 : 회원 탈퇴, 제제, 문의사항등 사이트 관리를 위한 별도의 관리자 페이지
